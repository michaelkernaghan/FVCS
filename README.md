# Fraser Valley Croquet Society Website

Official website for the Fraser Valley Croquet Society, a registered British Columbia society (S0075824) serving the Fraser Valley region.

## About

The Fraser Valley Croquet Society is a regional umbrella organization promoting and coordinating croquet throughout the Fraser Valley. As a registered BC society and member of Croquet Canada, FVCS provides organizational structure for member clubs and programs across the region.

## Website Structure

This is a static HTML website with the following pages:

- **Home** (`index.html`) - Society overview and registration information
- **About** (`about.html`) - History, governance, and organizational structure
- **Membership** (`membership.html`) - Information for clubs and programs joining FVCS
- **Programs** (`programs.html`) - Current croquet programs and activities
- **Contact** (`contact.html`) - Contact information and inquiries

## Design Philosophy

The website follows a formal, conservative, institutional design approach appropriate for a registered society. The design emphasizes:

- Professional credibility and trustworthiness
- Clear organizational hierarchy
- Accessibility and readability
- Formal color palette (navy, gold, neutral grays)
- Traditional serif typography for headings
